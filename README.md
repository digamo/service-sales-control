## âŒ¨ How To Use

### Clone Repository

```bash
$ git clone https://github.com/digamo/service-sales-control.git
```

### Run API

```bash

# Go to api folder
$ /salescontrol-api

# Run Aplication
$ mvn spring-boot:run test

# Swagger
http://localhost:8080/swagger-ui.html

```

### Run Web Project

```bash

# Go to web folder
$ /salescontrol-web

# Install Dependencies
$ npm install

# Run Aplication
$ ng server

```
Access url: `http://localhost:4200/`
